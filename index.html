Maze Maker

Maze Maker

This is a java program that uses a maze-generation algorithm in order to generate a maze. I added colors because I thought it looked cool, although they would not be in an actual generated maze.
The algorithm is based off of a pointer, which randomly moves through a grid without doubling back, and creates a path wherever it moves. When the pointer gets stuck, it moves backward until there is a new place to move.
This tool is not available on the web, and must be downloaded and compiled in order to use.

Wave Function Collapse

Wave Function Collapse

The wave function collapse algorithm is inspired by quantum mechanicsm, specifically superpositions and collapses. Each tile has a set of possibilities it can take up, and a number of rules dictating how the tiles can be placed in relation to each other. One tile is collapsed at random, by evaluating its state from the possibilities; the algorithm will then collapse the tile with the least possible states under the rules. This will continue until the entire grid is collapse.
This algorithm creates a grid of tiles with blue and green ends, with the rule that blue ends must connect to blue ends, and green ends must connect to green ends.
This tool is not available on the web, and must be downloaded and compiled in order to use.
